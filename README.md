# Http 완벽 가이드 스터디

<h3> 진행 방식 </h3>


* 사다리를 타기로 그날의 <b>진행자</b> 결정
* 진행하면서 토픽과 관련된 내용들을 토론하고, 서로 질문이 있거나 추가로 정리한 내용 발표
* 추가로 알았으면 좋겠다는 부분은 진행자가 준비
* 진행자가 서기역할도 담당
